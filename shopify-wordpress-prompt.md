# Prompt premium para Shopify / WordPress

Disena una pagina web ecommerce premium y funcional para MISTIKO PERFUMERIA, una marca de perfumes, insumos y kits de emprendimiento ubicada en Palmira, Valle del Cauca, Colombia.

Estilo visual:
Dubai Luxury + Rolex + Tom Ford + perfumeria arabe premium.

Paleta:
- Negro matte: #0A0A0A
- Dorado metalico: #D4AF37
- Rojo vino: #6B0000
- Blanco humo: #F5F5F5

Tipografias:
- Titulos: Cinzel
- Texto: Montserrat

Objetivo:
Vender perfumes premium, insumos y kits de emprendimiento en Palmira mediante WhatsApp, generando confianza, lujo visual y conversion rapida.

Estructura exacta del home:
1. Header fijo premium con logo Mistiko, menu y boton "Contactanos".
2. Hero cinematic con fondo oscuro de perfume, humo, oro y texto "El lujo tiene aroma".
3. Seccion de productos con tarjetas para Perfume Masculino Premium, Perfume Femenino Premium, Perfumes Arabes y Kit Para Emprender.
4. Botones "Agregar pedido" que abran carrito o checkout por WhatsApp.
5. Seccion "Por que funciona" explicando el modelo de perfumeria premium.
6. Seccion de kits: Kit Gold, Kit Black y Kit Royal.
7. Seccion de sedes en Palmira, asesoria por WhatsApp y pago confirmado.
8. Beneficios: margen atractivo, desde casa, alta rotacion, imagen premium, acompanamiento y recompra.
9. Testimonios premium con fondo negro y detalles dorados.
10. FAQ con preguntas sobre experiencia, kits, ganancias, ubicacion y contacto.
11. CTA final: "El momento perfecto para empezar es ahora".
12. Footer con WhatsApp, ubicacion en Palmira y navegacion.

Requisitos de diseno:
- Fondo negro profundo.
- Bordes dorados finos.
- Efecto gold foil.
- Tarjetas glassmorphism.
- Hover elegante.
- Animaciones suaves.
- Boton fijo movil para WhatsApp.
- Diseno responsive para celular.
- Mucho espacio visual.
- Nada colorido ni saturado.
- Sensacion de marca cara, misteriosa, elegante y confiable.

Copy principal:
Hero headline: "El lujo tiene aroma"
Subheadline: "Perfumes premium y kits para emprender en Palmira. Compra por WhatsApp, recibe asesoria personalizada y elige aromas con presencia para vender, regalar o usar todos los dias."
Botones: "Ver productos" y "Comprar por WhatsApp"

CTA final:
"Compra perfumes en Palmira o valida el kit ideal para empezar tu negocio. Un asesor de Mistiko confirma disponibilidad, entrega y forma de pago por WhatsApp."

Flujo de compra:
El usuario agrega productos o kits al pedido y envia el resumen por WhatsApp al numero 322 4929886. Los precios, aromas disponibles, entrega y forma de pago se confirman por WhatsApp antes del pago.
